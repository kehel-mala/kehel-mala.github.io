# Painting images

Drop painting image files in this folder (e.g. `untitled-i.jpg`), then reference
them from `src/paintings.ts` as `/paintings/<filename>`.

- Filenames are case-sensitive on GitHub Pages — keep them lowercase, no spaces.
- Any entry in `src/paintings.ts` whose image file is missing shows a labelled
  placeholder tile automatically, so the layout never breaks.
- Recommended: portrait-ish JPGs around 1000–1600px on the long edge for crisp,
  fast-loading thumbnails.
